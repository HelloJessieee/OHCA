Data Sources & Limitations

Data Sources

Generated Data

  • Population & Housing: Simulated data for 20 subzones
  • Socioeconomic Indicators: Modeled based on typical patterns
  • OHCA Cases: Generated using risk-based distribution
  • Volunteer Data: Hypothetical assignments for demonstration

Data Quality

This is a demonstration using simulated data. Real implementation would use actual Singapore demographic and health data.

Limitations & Methodology

Key Limitations

  • Data is simulated for demonstration purposes
  • Real OHCA data would require privacy protection
  • Geographic boundaries are simplified
  • Volunteer availability is estimated

Methodology

  • Risk scoring based on demographic factors
  • Volunteer assignment using proximity matching
  • AED placement optimization using coverage analysis
  • Forecasting using trend analysis
